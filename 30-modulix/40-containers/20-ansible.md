# Ansible EE Image

## Purpose

Execution Environment image used to run Ansible playbooks and collections for ModuLix.

## Image

- `quay.io/l-it/ee-wunder-ansible-ubi9`

## Official repository

- `https://github.com/lightning-it/container-ee-wunder-ansible-ubi9`

## Notes

- Contains Ansible runtime and collection dependencies required for playbook execution.
- Used by the runtime wrapper during `modulix-launcher` and `ansible-nav run` flows.
